Trophy Badge Assets
===================

This folder is optional in V1.

If you add PNG badge artwork, the app will load it automatically before using
the generated fallback icons.

Supported filename priority:
1. `<trophy_code>.png` (best, code-specific)
2. `<category>_<rarity>.png`
3. `<category>.png`
4. `default.png`

Examples:
- `first_maze_completed.png`
- `mini_game_common.png`
- `mini_game.png`
- `default.png`

Recommended export size:
- 128x128 (square PNG, transparent background)

The trophies page scales icons down for list display.
